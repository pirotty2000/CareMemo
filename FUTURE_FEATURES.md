# 今後の検討機能 (Future Features)

現在の健康管理メモ帳（CareMemo）の基盤が完成した段階で、次に「実用性」や「付加価値」を高めるために検討すべき機能のリストです。

## 1. レポート出力（PDF/画像）
- **目的**: 医師や家族へのスムーズな情報共有。
- **内容**: 指定した期間（例：直近1ヶ月）の血圧グラフや所見メモを、きれいにレイアウトされたPDFや画像として出力する機能。

## 2. セキュリティ強化（生体認証）
- **目的**: デリケートな個人情報の保護。
- **内容**: アプリ起動時や重要操作時に、Android標準の指紋認証や顔認証を求める機能。

## 3. 異常値のハイライト・通知
- **目的**: 利用者の体調変化の早期発見。
- **内容**: 設定した閾値（例：血圧が一定以上など）を超えたデータが入力された際、リスト上で強調表示や警告を行う。

## 4. 音声入力（Speech-to-Text）の統合
- **目的**: 入力負荷の軽減と情報の充実。
- **内容**: 所見メモの入力欄などで音声入力を可能にし、現場での素早い記録を支援する。

## 5. 写真による「視覚的記録」
- **目的**: 言葉で説明しにくい症状（皮膚の状態、食事、リハビリ等）の客観的記録。
- **内容**: 各レコード（特に所見メモ）に写真を添付できるようにする。

## 6. 一覧画面でのサマリー表示
- **目的**: 詳細画面に入らずに全員の「今の状態」を把握する。
- **内容**: 利用者一覧の各カードに、直近の血圧値や最後のメモの冒頭を小さく表示する。
