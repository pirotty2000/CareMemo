# CareMemo（ケアメモ）とは？
- ケアマネジャーなど居宅系の福祉従事者向けに、利用者のバイタルを入力・記録するアプリです。
- 自宅に訪問したときに測定・確認した情報や、報告書や意見書から得られた情報を記録しておくと、
  いつでもスマホやタブレットから記録の確認ができるようになります。
- 記録したデータはグラフで表示したり、値の判定(BMIなら低体重、など)を自動的に行ったり、
  レポートに整形して共有することができるので、データの分析・アセスメントに活用できます。

# 主な機能
- 入力されたデータの履歴を一覧で表示します。
- 数値データは、折れ線グラフで表示することもできます。
- メモを記録したり、メモに写真を添付して残したりできます。
- 保存された記録をレポートに加工し、PDFファイルとして出力し、メールなどに添付することができます。

# 記録できる情報の種類
5つのカテゴリの情報を入力・記録・参照できます。
## 1：身長・体重
- 身長と体重を記録し、一覧／グラフ表示します。
- BMIを自動で計算し、低体重などの判定を表示します。
## 2：バイタル
- 血圧(上)・血圧(下)・脈拍・体温を記録し、一覧／グラフ表示します。
- 高血圧／低血圧、頻脈／徐脈、低体温／発熱を判定し表示します。
## 3：血糖値・HbA1c
- 血糖値とHbA1cを記録し、一覧／グラフ表示します。
- 低血糖／高血糖、正常／正常高値／糖尿病予備群／糖尿病型を判定し表示します。
## 4：所見メモ
- 訪問時の様子などをメモして記録できます。
- 音声入力機能が使えるので、音声でメモを入力することができます。
- 写真を撮影しメモに添付することができます（最大3枚まで）
## 5：服薬管理
- 訪問時に確認した服薬状況を記録し、一覧／カレンダー表示します。
- 「朝・昼・夕・寝る前」毎に飲み忘れていたのか、声かけや介助で飲ませたのか、自身で服用していたのか記録できます。

# セキュリティ対策
このアプリは個人情報（氏名・生年月日）や個人の状態（血圧や血糖値など）を取り扱うため、
セキュリティには十分配慮して設計・開発しています。

- 記録されたデータはスマホ／タブレット内に暗号化されて保存されています。
- 写真を撮影すると、このアプリしか読み書きできない領域に保存されます
  （他のアプリやフォトビューアなどで見ることができません）。
- アプリ起動時に生体認証を求めさせることができます。
  この機能をオプションで有効にすれば、指紋認証など認証をパスしなければ
  起動しません。アプリを閉じて、復帰したときも認証が必要になっています。
  生体認証に対応していないデバイスでも、ジェスチャーなどでロックを解除
  しなければ起動できないようにできます。
- 記録したデータをレポートに整形しPDFファイルにする機能がありますが、
  PDFファイルにパスワードを設定することができます。
- データのバックアップ・リストア機能がありますが、バックアップしたデータに
  パスワードを設定することができます。
- 画面のキャプチャができない・キャプチャしても真っ黒になるように設定されています。
- スマホ・タブレットのタスク切り替え(画面右下の□)をタップしても、
  このアプリの画面だけ真っ白になり、画面に表示されている情報が見えないようになっています。
- このアプリでは利用者の氏名を扱いますが、オプションで伏せ字表示にすることができます。
