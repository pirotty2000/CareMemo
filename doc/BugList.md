# (1)：所見メモの写真

- ui\screens\UnifiedRecordScreen.kt
- ui\screens\ConditionDetailScreen.kt
- ui\screens\ConditionPhotoPreviewScreen.kt
- ui\screens\ConditionFullScreen.kt
- utils\ImageUtils
- viewmodel\PersonDetailViewModel



## 現象
所見メモで写真を撮影しても、登録されない

## 手順
1. 所見メモ一覧から一つタップして所見詳細へ
2. 所見詳細で右上のアイコンをタップして編集モードへ
3. 撮影ボタンをタップして写真撮影→写真のサムネイルが登録され撮影が成功した様子
4. 保存ボタンをタップして所見詳細に戻ると「写真がありません」になりサムネイルもない
5. 所見一覧に戻っても、記録者の左に「カメラ」アイコンがでない

## バックアップデータに写真が含まれていた場合
1. 所見メモ一覧から一つタップして所見詳細へ
2. 所見詳細で右上のカメラアイコンをタップして編集モードへ
3. 写真が3枚あれば、ごみ箱アイコンで削除→削除されなかった写真はサムネイルに残っている
   > ここで、撮影をせずに所見詳細に戻っても、一つしか削除しなかったのに「写真はありません」になってしまい、所見一覧に戻っても記録者にはカメラアイコンは表示されず、二度と写真をみることができない
4. 写真撮影ができるようになったので撮影ボタンをタップ
5. 以下、上記の「手順」同様の状況で、全ての写真が見えなくなる。
