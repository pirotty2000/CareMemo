# CareMemo（ケアメモ） ユーザマニュアル

## 1. はじめに
「CareMemo」は、ケアマネジャー、看護師、ヘルパーなどの専門職が、日々の訪問・介護現場で利用者の状態を素早く記録し、経過を把握するためのアプリです。

## 2. メイン画面（利用者一覧）
アプリを起動すると、登録されている利用者の一覧が表示されます。

*   **利用者を探す：**
    *   画面上の「所見メモ検索」欄に文字を入力すると、過去の記録内容（「転倒」「発熱」など）から利用者を絞り込めます。
    *   検索欄の下にある「あ」「か」「さ」…というボタン（カナ行）をタップすると、その行の苗字の利用者にジャンプします。
*   **記録を開始する：**
    *   一覧から利用者の名前をタップすると、記録するカテゴリ（バイタル、食事など）の選択画面が開きます。
*   **新規登録：**
    *   画面右下の「＋」ボタンをタップすると、新しい利用者を登録できます。

## 3. 利用者の登録と編集
### 新規登録・編集
1.  **名前・ふりがな：** 姓と名を分けて入力します。
2.  **識別用メモ：** 同姓同名の方がいる場合などに、「〇〇町内会」「〇〇施設」のように入力しておくと一覧で区別しやすくなります。
3.  **生年月日：** 「昭和」「平成」などを選択し、数字を入力します。
    *   ※入力欄の「年」を2桁（例：60）入れると、自動的に「月」の入力欄へ移動します。

### 利用の終了（削除）
*   利用者の右側にある「︙」ボタンから「利用を終了する」を選びます。
*   ※間違えて消してしまった場合は、画面下に表示される「元に戻す」をタップすると復旧できます。

## 4. 記録の入力
利用者を選択し、記録したい項目（バイタル、食事、排泄、所見メモなど）を選んで入力します。

*   **キーボード操作：** 「次へ」ボタンを押すと、次の入力欄にスムーズに移動できます。
*   **音声入力の活用：** 「所見メモ」では、スマホの音声入力機能を使うと便利です。句点（。）などは自動で補完されるよう調整されています。

## 5. グラフと履歴の確認
詳細画面では、これまでの記録がグラフやリストで表示されます。

### グラフの操作方法（拡大・縮小）
グラフの細かい動きを確認したいときは、以下の操作が可能です。
*   **横に広げる：** 画面に指を2本置き、指の間隔を広げるように動かすと、時間軸が拡大されます。
*   **横に縮める：** 2本の指で画面をつまむように間隔を狭めると、全体が表示されます。
*   **動かす：** 指1本で左右にスライドさせると、表示する時間を移動できます。
*   **元に戻す：** グラフをトントンと2回素早く叩く（ダブルタップ）と、最初の表示サイズに戻ります。

## 6. 設定とセキュリティ
画面右上のメニュー「︙」から「設定」を開けます。

*   **名前のマスキング：** 公共の場などで画面を見られたくない場合、名前を「山〇 太〇」のように伏せ字にできます。
*   **起動ロック：** 指紋認証や顔認証を設定しているスマホでは、アプリ起動時に認証を求めるように設定でき、個人情報を守ります。

## 7. レポート（PDF）出力
記録したデータは、サービス報告書などに活用できるようPDF形式で書き出すことができます。
*   データがない項目は自動的に除外され、すっきりとしたレイアウトで出力されます。
